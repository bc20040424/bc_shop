本项目以2025寒假字节青训营大项目抖音商城为基础进行编写，整个项目的源代码由我本人编写开发。
基本完成了以下几个功能：
  1.用户服务：包括：创建用户，登录，用户登出，删除用户，更新用户，以及获取用户身份信息；
  2.商品服务：包括：创建商品，修改商品信息，删除商品，查询商品信息（单个商品，批量商品）；
  3.购物车服务：包括：创建购物车，清空购物车，获取购物车信息；
  4.订单服务：包括：创建订单，修改订单信息，订单定时取消；
目前待完善的功能：
